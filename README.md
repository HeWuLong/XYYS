##  项目简介

星瀚影视是一款专为个人打造的私有化、全平台在线影院系统。项目采用前后端分离架构，通过深色沉浸式 UI 设计与自适应布局，力求在 web 端提供媲美原生 APP 的观影体验。系统集成了完整的鉴权体系与灵活的影视检索功能。

##  三大核心亮点

### 1. 自适应沉浸式黑夜 UI (Adaptive Dark UI)

- **设计理念**：采用低亮度的“深夜模式”配色体系（#111114），减少长时间观影的视觉疲劳。
- **自适应布局**：利用 Flex 与 Grid 布局，完美适配移动端、平板与 PC 宽屏，确保在各种终端下均能实现全屏无缝观影。

### 2. JWT 无状态鉴权体系

- **安全性**：基于 Spring Security + JWT 实现。服务端不存储 Session，降低内存开销，同时支持跨域扩展。
- **双重保护**：集成 `JwtAuthenticationFilter` 对每个请求进行无感拦截与令牌校验，确保私密资源仅供授权用户访问。

### 3. 多维动态复合检索

- **查询引擎**：基于 MyBatis-Plus 的 `LambdaQueryWrapper` 构建动态 SQL。
- **灵活组合**：支持“类型+地区+年份+排序”的任意维度组合过滤，实现毫秒级的响应速度。

##  技术栈

### 前端

- **框架**：uni-app (Vue 3.0 + Composition API)
- **网络**：Promise 封装的通用 `request.js`
- **样式**：SCSS / CSS3 (基于 `rpx` 与 `vh/vw` 的自适应方案)

### 后端

- **核心**：Spring Boot 3.x
- **持久层**：MyBatis-Plus (高性能 SQL 封装)
- **数据库**：MySQL 8.0
- **安全**：Spring Security + JSON Web Token

##  🗺️ 演进路线图 (Roadmap)

本项目致力于从单一的流媒体播放器，逐步演进为**全场景的个人数字聚合生态 (Personal Digital Ecosystem)**。后续开发将严格遵守高内聚、低耦合的微服务化架构思想推进。

### 阶段一：影视流媒体基建 (Phase 1: Video System Refinement) - `🚀 进行中`
- [x] **核心框架搭建**：前后端分离架构，路由拦截与基础页面渲染成型。
- [x] **多维检索中枢**：基于 MyBatis-Plus 动态 SQL 实现毫秒级复合条件过滤。
- [ ] **解析引擎对接**：完成真实视频源（M3U8/MP4）解析与选集动态路由跳转。
- [ ] **足迹与状态同步**：设计 `vod_history` 表，实现多端（PC/Mobile）历史观看进度实时同步。
- [ ] **可视化 Admin 控制台**：开发第三方视频 API 数据采集、定时同步（Quartz）与配置面板。
- [ ] **高并发改造**：引入 Redis 缓存机制，实现“实时播放热度榜”及恶意接口防刷（Rate Limiting）。

### 阶段二：统一数字基座 (Phase 2: Unified Platform Base) - `⏳ 规划中`
- [ ] **前端工程解耦**：采用 uni-app 分包策略 (SubPackages)，将系统物理隔离为 `pages_video`、`pages_music` 和 `pages_im`，保障首页秒开率。
- [ ] **SSO 统一认证中心**：升级 JWT 无状态鉴权机制，支持单点登录，并接入第三方（微信/QQ/GitHub）OAuth2.0 快捷登录。
- [ ] **全局控制看板 (Dashboard)**：打造全新的个人主页导航入口，汇聚影音时长、社交动态、系统资源监控的数据可视化看板（ECharts）。

### 阶段三：星瀚随身听 (Phase 3: Music Streaming Module) - `⏳ 规划中`
- [ ] **混合音频引擎**：接入全局悬浮音频播放器，支持 Web Audio API 音效处理、后台保活与锁屏控制。
- [ ] **沉浸式视听 UI**：解析 Lrc 格式文件实现精准歌词滚动，基于封面色彩提取（Color Thief）实现背景动态高斯模糊渐变。
- [ ] **私有云曲库管理**：支持用户自建歌单、无损音质（FLAC/WAV）在线解码解析，结合开源 API 抓取全网元数据补充。

### 阶段四：即时通讯中枢 (Phase 4: Instant Messaging) - `⏳ 规划中`
- [ ] **长连接架构**：基于 `Spring Boot` + `Netty` (或原生 WebSocket) 构建低延迟、高可用的双向通信通道。
- [ ] **实时路由分发**：利用 Redis Pub/Sub 与状态缓存，实现高并发下的在线状态心跳监听与多节点消息路由。
- [ ] **富文本聊天引擎**：支持单聊/群聊，支持图片、语音（WebRTC 录音转码）等多媒体消息的高效 OSS 存储与传输。

### 阶段五：AI 智能引擎 (Phase 5: AI & Smart Tech) - `🧠 探索中`
- [ ] **多维用户画像**：记录观影类型、听歌频次、搜索偏好，构建轻量级用户行为标签图谱。
- [ ] **智能推荐算法**：基于协同过滤 (Collaborative Filtering) 与余弦相似度算法，实现千人千面的“猜你喜欢”瀑布流推荐。
- [ ] **LLM 助理集成**：探索接入大语言模型 API，提供智能影视剧情问答与找歌助手功能。

### 阶段六：容器化与 DevOps (Phase 6: CI/CD & Deployment) - `☁️ 展望`
- [ ] **Docker 容器化**：编写 Dockerfile 与 docker-compose.yml，实现前后端中间件（MySQL/Redis）的一键快速编排部署。
- [ ] **持续集成**：利用 GitHub Actions 实现代码推送后的自动化构建与打包部署。
- [ ] **全链路监控**：接入 Spring Boot Admin 或 Prometheus + Grafana，监控线上 CPU、内存及 JVM 运行状态。
